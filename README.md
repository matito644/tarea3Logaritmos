## Algoritmos probabilísticos

### Compilar
Para compilar los códigos basta ejecutar:
```
g++ -O3 archivo.cpp -o run
```
Y luego:

```
./run
```

Lo anterior estando en la carpeta que contiene todos los códigos.

Versión de c++: 'C++17'.
